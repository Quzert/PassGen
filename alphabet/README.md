Папка `alphabet` содержит предустановленные наборы символов (алфавиты) для генерации паролей.

Файлы: (каждый файл — один алфавит, все символы подряд в одной строке)
- `default.txt` — буквы верхнего/нижнего регистра, цифры и спецсимволы
- `no-special.txt` — буквы + цифры (без спецсимволов)
- `lowercase-digits.txt` — только маленькие буквы и цифры
- `lowercase.txt` — только маленькие буквы
- `digits.txt` — только цифры

Как добавить свой алфавит
1. Создайте новый файл в этой папке, например `my-alphabet.txt`.
2. Впишите в него в одну строку все символы, которые хотите использовать, например:

   abcdefghijklmnopqrstuvwxyz0123456789@#

3. В вашем приложении или скрипте при необходимости читайте содержимое файла и передавайте строку в функцию генерации пароля.

Пример (bash):

```bash
ALPH=$(cat alphabet/my-alphabet.txt)
./build/passgen "master" "site" 16 -a "${ALPH}"
```

Примечание
- Файлы в этой папке не требуют изменений в исходном коде: добавляйте/редактируйте файлы по мере нужды.
- Убедитесь, что алфавит не пустой; иначе генератор может бросить исключение.
